你肯定也遇到过：AI 看起来什么都能干，但真用起来总差那么点意思。不是你不会用，是你还不了解它的脾气。这个专栏不讲公式，用生活里的概念讲清楚模型、Agent、提示词到底怎么回事。让你看完知道什么场景该用 AI、怎么跟它配合、做到什么程度算靠谱。

## 阅读建议

建议按顺序读：

- 先理解模型为什么能生成看似合理的回答（第 1 篇）。
- 再理解 Agent 如何把模型能力放进目标、工具、权限和验证闭环里（第 2 篇）。
- 然后理解 AI 每次回答到底"看到了什么"，以及为什么会忘、会跑偏（第 3 篇）。
- 再理解 Tool、Skill 和 MCP 如何让 AI 从"会回答"走向"会干活"（第 4 篇）。
- 理解提示词的任务契约本质（第 5 篇）。
- 用 OpenCode 跑通第一个实操闭环（第 6 篇）。
- 最后理解怎么帮 AI 设计记忆，让它记住你做过的事（第 7 篇）。
- 理解 AI 怎么靠执行系统完成复杂任务：ReAct 循环、任务拆解、工具调用、出错处理（第 8 篇）。
