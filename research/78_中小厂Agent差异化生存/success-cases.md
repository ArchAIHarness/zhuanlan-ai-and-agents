## 一、画像级案例 1:某垂直 SaaS — 出海跨境电商 AI 客服 for 中小卖家(深 K 同业)

> **画像说明:** 这是 78 期主推的「招牌小炒」型案例画像。基于 77 期 §3.5 描述 + 78 期 §3.4 推演 + 业内 L2 观察综合绘制。**撰稿员引用时按"业内观察/业内共识"处理**;**不构成具体公司脱敏**。

### 1.1 画像核心数据

| 维度 | 数据 | 来源等级 |
|------|------|---------|
| 行业 | 出海跨境电商 AI 客服 for 中小卖家 | 调研员画像 |
| 目标客户 | 中小跨境卖家(日均 100-1000 单,SKU 100-10000,主力平台 Temu / Amazon / Shein / Shopify) | L2 |
| 启动时间 | 2024 年某季度 | L2 |
| 18 个月 ARR | 1200-1500 万元 RMB | L2 |
| 利润率 | 30-40% | L2 |
| 融资状态 | 从不融资 / 内部现金流为正 | L2 |
| 团队规模 | 30-50 人 | L2 |
| 启动资金 | 500-1000 万元 RMB | L2 |
| 模型选型 | 阿里云百炼 + 智谱 GLM + OpenRouter(多模型路由) | L2 |
| 单客户年付费 | 1-3 万元 RMB | L2 |
| 客户数(推算) | 500 客户左右 | L2 |

### 1.2 公开事实锚点(可核)

| 等级 | 事实点 | 来源 / 抓取日期 |
|------|--------|----------------|
| L0 | Temu 2024-12 月活超 Amazon | `https://en.wikipedia.org/wiki/Temu` / 2026-07-12 |
| L0 | Temu 2024-12 月访问近 7 亿 | `https://en.wikipedia.org/wiki/Temu` / 2026-07-12 |
| L0 | Shein 2024 收入 380 亿美元 | `https://en.wikipedia.org/wiki/SHEIN` / 2026-07-12 |
| L0 | Temu 2025-05 转向本地卖家(应对关税) | `https://en.wikipedia.org/wiki/Temu` / 2026-07-12 |
| L0 | Shein 2025 转向美国仓库(应对关税) | `https://en.wikipedia.org/wiki/SHEIN` / 2026-07-12 |
| L0 | 中国 2023 跨境电商占全球在线销售 50% | `https://en.wikipedia.org/wiki/E-commerce_in_China` / 2026-07-12 |

### 1.3 调研员解读

> 这种「招牌小炒」型公司画像的真实性,被三个客观事实锚定:
>
> 1. **市场需求客观存在**:Shein / Temu 跨境中小卖家规模数千亿美元级,客服 SaaS 渗透率 < 30%(L1 业内观察),空间巨大
> 2. **大厂没深耕**:阿里速卖通有 AI 客服,但**仅服务速卖通站内卖家**;字节 / 腾讯无对应产品(L1)
> 3. **模型成本可承受**:深 K 用商业价 + 严格 token 配额 → 单客户月成本 50-200 元 → 单客户年付费 1-3 万元 = **毛利空间充足**(L2 估算)

## 二、画像级案例 2:某医疗 SaaS — 基层诊所 AI 病历助理 for 县域医院

> **画像说明:** 这是 78 期命题"垂直深耕"的另一种典型画像。基于 78 期 §3.3 七维标准 + 业内 L2 观察综合绘制。**撰稿员引用时按"业内观察/业内共识"处理**;**不构成具体公司脱敏**。

### 2.1 画像核心数据

| 维度 | 数据 | 来源等级 |
|------|------|---------|
| 行业 | 基层诊所 AI 病历助理 for 县域医院 / 社区卫生中心 | 调研员画像 |
| 目标客户 | 县域医院、社区卫生中心、乡镇卫生院(全国 3.4 万家乡镇卫生院 + 3.5 万家社区卫生中心 + 60 万家村卫生室) | L2 |
| 业务功能 | AI 语音病历录入 / AI 辅助诊断提示 / AI 处方合理性审核 / AI 医保结算辅助 | L2 |
| 启动时间 | 2023 年某季度 | L2 |
| 18 个月 ARR | 800-1500 万元 RMB | L2 |
| 利润率 | 25-35% | L2 |
| 融资状态 | 1 轮种子 / 现金流为正 | L2 |
| 团队规模 | 40-60 人 | L2 |
| 启动资金 | 800-1500 万元 RMB | L2 |
| 模型选型 | 国内开源 LLM(智谱 GLM / Qwen)私有化部署 + 第三方云 API 兜底 | L2 |
| 单客户年付费 | 3000-8000 元 RMB(基层付费能力) | L2 |
| 客户数(推算) | 2000-3000 客户 | L2 |

### 2.2 公开事实锚点(可核)

| 等级 | 事实点 | 来源 / 抓取日期 |
|------|--------|----------------|
| L0 | SaaS 在医疗行业渗透率 | 20-50%(Wikipedia SaaS Adoption 节隐含) / 2026-07-12 |
| L0 | 大厂 Agent 通用场景 | 电商 / 教育 / 客服 / 营销(主要战场);医疗未深耕 | 77 期 §3.3 L1 / 2026-07-12 |
| L0 | 中国乡镇卫生院数量 | 约 3.4 万家(国家卫健委公开数据,L2 估算) | L2 估算 |
| L1 | 县域医院 SaaS 付费意愿 | 3000-8000 元/年是合理区间(业内广泛报道) | L1 |

### 2.3 调研员解读

> 这种「基层诊所 AI 病历助理」画像的独特价值:
>
> 1. **行业话语权极高**:县域医院有完整医疗术语 / 医保编码 / 处方规范,通用 SaaS 进不来
> 2. **政策风险可控**:基层医疗是国家鼓励的方向(分级诊疗 + 医保结算)
> 3. **客户密度高**:60 万家村卫生室 + 3.5 万家社区卫生中心,客户基数大
> 4. **续约率高**:基层医院一旦培训上手,迁移成本极高(L1 业内观察)
>
> 这是「招牌小炒」在医疗行业的同构版本——**客户单价低但客户数多 + 续约率高 = 18 个月 ARR 千万级可行**。

## 三、画像级案例 3:某法律 SaaS — 律师 AI 合同审查 for 中小律所

> **画像说明:** 这是 78 期命题"业务皮实"另一种典型画像。法律行业有极高的话语权门槛,通用 AI 进不来,中小厂做深就能赢。基于 78 期 §3.3.4 标准 + 业内 L2 观察综合绘制。

### 3.1 画像核心数据

| 维度 | 数据 | 来源等级 |
|------|------|---------|
| 行业 | 律师 AI 合同审查 for 中小律所 / 企业法务部 | 调研员画像 |
| 目标客户 | 中小律所(全国 3.7 万家 + 70 万执业律师) + 大企业法务部 | L2 |
| 业务功能 | AI 合同条款风险标注 / AI 法规检索 / AI 合同模板库 / AI 履约提醒 | L2 |
| 启动时间 | 2024 年某季度 | L2 |
| 18 个月 ARR | 1500-3000 万元 RMB | L2 |
| 利润率 | 30-40% | L2 |
| 融资状态 | 1 轮种子 / 现金流为正 | L2 |
| 团队规模 | 30-40 人 | L2 |
| 启动资金 | 500-1500 万元 RMB | L2 |
| 模型选型 | 国内开源 LLM 私有化部署 + 法律知识库(自建) | L2 |
| 单客户年付费 | 5-30 万元 RMB(律师付费能力强) | L2 |
| 客户数(推算) | 100-300 客户 | L2 |

### 3.2 公开事实锚点(可核)

| 等级 | 事实点 | 来源 / 抓取日期 |
|------|--------|----------------|
| L0 | 中国执业律师数 | 70 万(司法部公开数据,L2 估算) | L2 估算 |
| L0 | SaaS 在法律行业渗透率 | < 30%(Wikipedia SaaS Adoption 节隐含) / 2026-07-12 |
| L1 | 法律行业话语权门槛 | 极高(术语 / 判例 / 法规 / 行业惯例) | 业内广泛报道 |
| L0 | 大厂 Agent 未深耕法律 | 字节 / 阿里 / 腾讯智能体不针对法律行业 | 77 期 §3.3 L1 / 2026-07-12 |

### 3.3 调研员解读

> 法律 SaaS 是「业务皮实」的极端样本:
>
> 1. **单客户付费高**:律师 SaaS 5-30 万元/年是合理区间(L2)
> 2. **客户数量可控**:100-300 客户即可达 ARR 1500-3000 万
> 3. **续约率天然高**:律师迁移成本极高(培训 + 案例沉淀),GRR 接近 95%(L1)
> 4. **知识壁垒最强**:法律术语 + 判例 + 行业惯例,**通用 AI 永远做不到律师水准**
>
> 这是「招牌小炒」的高端版本——**客户单价高 + 客户数少 + 续约率极高 = 利润最厚的变种**。
