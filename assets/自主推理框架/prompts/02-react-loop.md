# 02 ReAct 循环四步

## Style
Hand-drawn educational framework diagram on warm cream paper (#F5F0E8). Black sketch lines. Soft pastel blocks in orange, green, blue. Handwritten Chinese labels. Simple line icons.

## Content
A circular flow diagram showing the ReAct loop with 4 steps arranged in a clockwise cycle.

**Step 1 - 思考 (Think)**: Top-left. Icon: lightbulb or thought bubble. Color: soft blue. Label: "思考——我现在知道什么？下一步该做什么？"

**Step 2 - 行动 (Act)**: Top-right. Icon: hand doing something or gear. Color: soft orange. Label: "行动——执行一个具体动作"

**Step 3 - 观察 (Observe)**: Bottom-right. Icon: eye or magnifying glass. Color: soft green. Label: "观察——行动结果是什么？"

**Step 4 - 循环 (Loop)**: Bottom-left. Icon: circular arrow. Color: soft blue. Label: "带着新信息回到第一步"

Arrows connect the steps in a circle: Think → Act → Observe → back to Think. The arrows are sketch-style, hand-drawn.

Center of the circle: a small label "ReAct = Reasoning + Acting"

Below the diagram: "不是先想再做，是边想边做，做完看结果，看完接着想"

Use simple line art, hand-drawn sketch style for all elements.
