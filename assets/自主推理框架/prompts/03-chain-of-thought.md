# 03 思维链作为推理锚点

## Style
Hand-drawn educational framework diagram on warm cream paper (#F5F0E8). Black sketch lines. Soft pastel blocks in orange, green, blue. Handwritten Chinese labels. Simple line icons.

## Content
A visual explanation of Chain of Thought showing two paths.

**Top path (错误路径 - 跳步)**: A person jumping from a starting point directly to a conclusion, with a big red X. The path is straight but broken. Label: "直接跳到结论 → 容易跑偏"

**Bottom path (正确路径 - 不跳步)**: A person stepping on visible stones across a river. Each stone has a label: "已知条件" → "第一步推理" → "第二步推理" → "中间结论" → "最终答案". The stones are solid pastel-colored blocks. Green check marks along the path. Label: "每一步写出来 → 每一步都是下⼀步的锚点"

**Bottom caption**: "思维链的本质不是'解释'，是'不跳步'"

Use simple line art for figures (silhouettes only, no faces), sketch-style stones/blocks for the stepping stones.
