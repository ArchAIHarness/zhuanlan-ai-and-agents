# 03: FDE 装备清单

## Style
Hand-drawn educational framework diagram on warm cream paper (#F5F0E8). Black sketch lines (#2C2C2C), soft pastel blocks. No dark backgrounds, no 3D, no realistic faces.

## Composition
Square 1:1. An open toolbox icon. Inside two compartments. Left compartment labeled "思考装备" with icons: DDD (boundary), 事件风暴 (lightning bolt), SDD (target), 业务建模 (puzzle piece), color soft blue (#7BA7C9). Right compartment labeled "干活装备" with icons: code bracket, MCP plug, glue code (link chain), deploy rocket, color sage green (#85BAA1). Below toolbox: small text "FDE 装备清单".
