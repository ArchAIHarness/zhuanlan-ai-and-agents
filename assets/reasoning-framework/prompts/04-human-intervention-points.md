# 04 人工接管点的三种类型

## Style
Hand-drawn educational framework diagram on warm cream paper (#F5F0E8). Black sketch lines. Soft pastel blocks in orange, green, blue. Handwritten Chinese labels. Simple line icons.

## Content
Three columns showing three types of human-in-the-loop intervention points.

**Left column - 确认站**: A human figure (silhouette, no face) and a robot/computer face looking at each other with a question mark between them. Color: soft orange. Description below: "AI不确定时，停下来问你。需要做关键决策但信息不足。"

**Center column - 审批站**: A document/paper with a "审批" stamp on it, human figure giving a thumbs up. Color: soft blue. Description below: "AI完成重要步骤后，让你确认再继续。比如文案草稿、代码审查。"

**Right column - 失败回退**: A robot with a small white flag or hands up, a human figure stepping in. Color: soft green. Description below: "AI搞不定时主动认输。API不通、权限不够、超出能力范围。"

**Top header row**: "人工接管点的三种类型" in handwritten Chinese.

**Bottom**: "在正确的地方设卡，不要在每一步都插手。"

Use simple line art, sketch style. No realistic faces.
