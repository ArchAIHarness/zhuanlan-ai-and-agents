## publisher 操作提示

### CSDN 工作流(按 §七.4)
1. 正文注入:用 `execCommand('insertText')` 注入完整 Markdown(平台适配版去掉 H1)
2. 配图必须在发布设置之前插入:每张按"定位光标 → 点击图片按钮(CDP 鼠标事件,不是 Playwright click)→ setInputFiles 上传 → 等待 CDN URL 插入"操作
3. 发布设置合法顺序:封面图 → 标签 → 摘要 → 分类专栏 → 文章类型 → 可见范围
4. 最终状态只能是「草稿」,不得点"发布文章""定时发布"或"确定并发布"等任何将内容公开的按钮
5. 图片按钮必须通过 CDP 鼠标事件(Chrome DevTools MCP `click`);Playwright `.click()` 会被 Vue 拦截,不可用
6. 完成后告知主编:草稿已就绪 + 草稿 ID(draftId)

### 掘金工作流(juejin-publisher-v2 skill)
1. 正文注入同 CSDN(平台适配版去掉 H1)
2. 配图位置同 CSDN
3. 发布设置:封面 → 标签 → 摘要 → 分类 → 文章类型 → 可见范围
4. 最终状态 = 草稿
